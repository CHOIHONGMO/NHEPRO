Component: Select

EVF.Select

Select 컴포넌트 API
Properties:
Name Type Description
useMultipleSelect boolean 다중선택 가능여부를 지정합니다.
searchType string 다중선택 시 문자열 검색방법을 지정합니다. (A:좌우 매칭, L:좌측기준 매칭)
delimeter string 다중선택된 값들의 구분자를 지정합니다.
기본값은 ","(콤마)이지만 Option의 값에 콤마가 들어간 경우를 위한 속성입니다.
Since:
  • 1.0.0
Example
<e:select id="${id}" name="${name}" useMultipleSelect="false" ... />
