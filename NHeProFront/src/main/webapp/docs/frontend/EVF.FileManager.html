Component: FileManager

EVF.FileManager

파일의 업로드와 다운로드를 담당하는 컴포넌트입니다. STOCATCH 테이블에서 업로드된 파일정보를 관리하며, 파일ID (키), 그리고 파일ID의 시퀀스(해당 키로 묶인 파일들)로 구성되어 있습니다.
Properties:
Name Type Description
downloadable boolean 다운로드의 가능/불가에 대한 초기값을 설정합니다.
uploadable boolean 업로드의 가능/불가에 대한 초기값을 설정합니다.
onSuccess string 업로드 성공 시 호출할 콜백함수명을 설정합니다.
onBeforeRemove string 업로드한 파일을 삭제하기 전에 호출할 콜백함수명을 설정합니다.
onAfterRemove string 업로드한 파일을 삭제한 후에 호출할 콜백함수명을 설정합니다.
onError string 업로드 시 오류가 발생하면 호출할 콜백함수명을 설정합니다.
onFileAdd string 업로드 큐에 파일등록 시 호출할 콜백함수명을 설정합니다.
onFileClick string 업로드한 파일 클릭 시 호출할 콜백함수명을 설정합니다.
fileId string 업로드한 파일정보를 파일매니저에 로드하기 위한 ID를 설정합니다.
required boolean Store에서 validate 체크 시, 최소 1개 이상의 파일을 업로드해야 하도록 제어하려면 true로 설정합니다.
isAutoUpload string 파일 선택 후 바로 업로드를 수행시키려면 true로 설정합니다.
