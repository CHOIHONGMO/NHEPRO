EVF.InputDate

날짜 입력 컴포넌트입니다.
Properties:
Name Type Description
format string 날짜포맷을 지정합니다. ui 태그의 dateFmt 속성을 무시합니다.
yyyy-mm-dd, yyyy/mm/dd, yyyy-mm, yyyy/mm 등의 포맷팅을 지원합니다.
minDate string 최소선택가능날짜값을 지정합니다. 현재 미구현 상태입니다.
maxDate string 최대선택가능날짜값을 지정합니다. 현재 미구현 상태입니다.
fromDate string 기간을 제어하기 위한 속성입니다. 현재 미구현 상태입니다.
toDate string 기간을 제어하기 위한 속성입니다. 현재 미구현 상태입니다.
onSelectDate string 달력에서 날짜 선택 시 호출할 콜백함수명을 설정합니다.
onEnter string 날짜폼에서 엔터 클릭 시 호출할 콜백함수명을 설정합니다.
