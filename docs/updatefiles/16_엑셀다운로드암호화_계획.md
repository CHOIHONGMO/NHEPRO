# 그리드 엑셀 다운로드 시 엑셀 암호화 적용 구현 계획서

그리드 데이터를 엑셀로 다운로드할 때 비밀번호 입력을 유도하는 공통 모달 팝업을 제공하고, 사용자가 4자리 숫자의 비밀번호를 입력한 경우 해당 비밀번호로 암호화된 엑셀 파일을 생성하여 다운로드하는 기능을 구현합니다. 
비밀번호를 입력하지 않고 확인을 누르면 암호화 없이 다운로드되며, 취소 시에는 다운로드가 중단됩니다.

---

## 1. 전제조건 및 제약사항
- **암호화 대상 확장자:** Excel 2007 이상 포맷 (.xlsx)
- **비밀번호 규칙:** 4자리 숫자 (`0000` ~ `9999`)만 입력 가능
- **동작 방식:**
  - 비밀번호 미입력 후 확인 클릭: 비밀번호 설정 없이 원본 파일 다운로드
  - 비밀번호 입력 후 확인 클릭: 해당 비밀번호로 암호화된 엑셀 파일 다운로드
  - 취소 클릭: 다운로드 차단
- **적용 대상:** 전사 공통 RealGrid 엑셀 다운로드 버튼 이벤트

## 3. 검증 계획

### 3.1 수동 검증 케이스
1. 임의의 화면에서 그리드 엑셀 다운로드 버튼 클릭 시 모달 팝업이 노출되는지 확인.
2. **테스트 1 (취소):** 모달에서 취소 클릭 시 파일 다운로드가 취소되는지 확인.
3. **테스트 2 (미입력):** 모달에서 아무것도 입력하지 않고 확인을 눌렀을 때, 비밀번호가 걸려있지 않은 정상적인 엑셀 파일이 즉시 다운로드되는지 확인.
4. **테스트 3 (유효성 오류):** 4자리 이외의 숫자, 문자를 입력한 경우 alert 메세지 노출 및 다운로드 동작 차단 확인.
5. **테스트 4 (암호화 성공):** 4자리 숫자(예: `5678`) 입력 후 확인 시 엑셀 다운로드가 실행되며, 다운로드된 파일을 열었을 때 정상적으로 비밀번호 입력을 요구하는지 확인.
