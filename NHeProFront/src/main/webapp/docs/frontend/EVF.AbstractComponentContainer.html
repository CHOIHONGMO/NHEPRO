Component: AbstractComponentContainer

EVF.AbstractComponentContainer

컨테이너 성격을 가진 컴포넌트들이 상속받아 구현하는 컴포넌트
Since:
  • 1.0.0

Extends

  • EVF.AbstractComponent

Methods

(static) iterateAll(func)

해당 컨테이너 내의 모든 컴포넌트들을 가져와서 순환함수를 호출합니다. iterator()는 입력컴포넌트만 순환합니다.
Parameters:
Name Type Description
func function 함수
Example
EVF.C('${searchPanelId}').iterateAll(function() {
   alert(this.getID());  // this 객체는 입력 컴포넌트 객체를 참조합니다.
});

(static) iterator(func)

해당 컨테이너 내의 입력 컴포넌트들에 대해 파라미터 함수를 호출합니다. iterateAll()은 모든 컴포넌트를 순환합니다.
Parameters:
Name Type Description
func function 함수
Example
EVF.C('${searchPanelId}').iterator(function() {/n
    alert(this.getValue());  // this 객체는 입력 컴포넌트 객체를 참조합니다.
});

(static) resetData()

해당 컨테이너 내의 모든 입력컴포넌트의 값을 초기화한다.
Example
EVF.C('${searchPanelId}').resetData();

(static) setJsonData()

JSON 데이터의 ID와 매칭되면 해당 JSON 데이터의 값을 컴포넌트의 값에 셋팅한다.
Example
EVF.C('${searchPanelId}').setJsonData({"TEXT_ID":"Hello, EVF!"});
