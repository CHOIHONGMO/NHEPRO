## 3. 검증 및 빌드 확인 사항
* JEUS 8 업그레이드 배포 후 `nhepro.properties` 파일 내의 `eversrm.log.path.*` 값을 신규 JEUS 8 로그 경로로 변경 후 기능(로그 뷰어)이 정상 동작하는지 테스트가 필요합니다.
* 변경된 속성을 읽어오는지 WAS 기동 로그 및 로그 뷰어 동작 여부로 직접 확인 가능합니다.
